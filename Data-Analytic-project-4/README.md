# Data Visualization - Project 4

## 📌 Project Overview

This project demonstrates data visualization techniques using an E-commerce Sales dataset. Various charts were created to identify trends, patterns, and business insights.

## 🎯 Objectives

- Visualize sales data using charts.
- Understand customer purchasing behavior.
- Identify trends and patterns.
- Present insights through effective visualizations.

## 🛠 Tools Used

- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## 📊 Visualizations Created

- Bar Chart
- Pie Chart
- Line Chart
- Histogram
- Box Plot
- Scatter Plot
- Correlation Heatmap
